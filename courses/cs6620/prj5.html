Hardware Used and Other Details

Render times were measured using std::chrono::high_resolution_clock and only include time to render, ie. time to load the scene and write the images to disk is ignored. The image was rendered with one sample per pixel since it's incredibly slow without the BVH and I didn't want to wait too long.

CPU: Intel i5-2500K @ 4.0GHz, 4 hardware threads
RAM: 8GB 1600MHz DDR3
Compiler: gcc 4.9.1 x86_64 (on Windows, built by MinGW-W64 project)
Compilation Flags: -m64 -O3 -march=native -flto
